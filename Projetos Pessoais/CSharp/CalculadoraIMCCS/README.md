# CalculadoraIMCCS

Calculadora de IMC (Índice de Massa Corporal) em C#.

## Funcionalidades
- Cálculo do IMC
- Classificação do resultado

## Como usar
1. Compile com `dotnet run` ou `csc`
2. Informe peso e altura

## Estrutura do projeto
- `Program.cs`: Código principal
- `README.md`: Este arquivo
